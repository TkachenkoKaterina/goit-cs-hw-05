========================
Async File Sorter
========================

1. Генерація тестових файлів
----------------------------
Запустіть генератор:
    python gen_files.py

Він створить у папці `test_src` 30 випадкових файлів з різними розширеннями
(деякі без розширень) та підпапками.

2. Запуск сортувальника
------------------------
Приклад запуску з переміщенням у `sorted_out`:
    python async_file_sorter.py -s test_src -o sorted_out -v

Опції:
    -s / --source   — вихідна папка (рекурсивно читається)
    -o / --output   — цільова папка
    --workers       — кількість одночасних копіювань (стандартно 8)
    -v / -vv        — рівень логів: INFO / DEBUG

3. Результат
------------
У `sorted_out` будуть створені підпапки за розширеннями:
    txt/
    jpg/
    pdf/
    png/
    docx/
    _no_ext/

Файли копіюються з збереженням метаданих (час зміни тощо).
